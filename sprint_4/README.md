# Sprint 4

Sprint 4 builds an object-oriented multi-agent pipeline with research, writing, design, development, and review agents.

Files:
- `lesson_4_1_base_agent.py` — shared Agent base class
- `lesson_4_2_researcher.py` — ResearcherAgent using web search tools
- `lesson_4_3_writer.py` — WriterAgent for Markdown essays
- `lesson_4_4_designer.py` — DesignerAgent for structured JSON output
- `lesson_4_5_developer.py` — DeveloperAgent for HTML generation
- `lesson_4_6_reviewer.py` — ReviewerAgent for PASS/FAIL auditing
- `lesson_4_7_orchestrator.py` — central publication pipeline coordinator
